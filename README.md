# AGROIDEAS GxP: Sistema de Automatización de Gestión por Procesos

Este sistema es una herramienta avanzada para la sistematización, documentación y simulación de los procesos institucionales de AGROIDEAS (GxP), alineado a los estándares de modernización del MIDAGRI.

## 🚀 Características Principales

- **Inventario Maestro Jerárquico**: Gestión de procesos desde Nivel 0 (Macroprocesos) hasta Nivel 4 (Actividades/Tareas).
- **Editor de Fichas GxP**: Interfaz premium para la caracterización SIPOC y técnica de documentos institucionales.
- **Ingesta Normativa**: Procesamiento de datos masivos desde JSON con validación de integridad.
- **Arquitectura Local-First**: Persistencia robusta basada en LocalStorage para independencia de servidores externos.
- **Dashboard de Gestión**: Visualización en tiempo real del estado de la sistematización.

## 🛠️ Tecnologías

- **Frontend**: HTML5, CSS3 (BEM, Glassmorphism), JavaScript (Vanilla).
- **Metodología**: Gestión por Procesos (GxP) - Norma Técnica MIDAGRI.
- **Estado**: Fase 3 - Consolidación de Inventario y Documentación.

## 📂 Estructura del Proyecto

- `/datos`: Archivos JSON maestros.
- `/js`: Controladores logicos (Storage, Core Engine, Admin).
- `/css`: Diseño institucional y tokens visuales.
- `index.html`: Dashboard principal.
- `documentos-procesos.html`: Repositorio documental.
- `inventario.html`: Explorador jerárquico.
